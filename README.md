# Continuous-Time Data-Driven Control

ecc2025_algorithm1, ecc2025_algorithm2: MATLAB code for the numerical examples of the paper: A. Bosso, M. Borghesi, A. Iannelli, G. Notarstefano, A. R. Teel, "Derivative-Free Data-Driven Control of Continuous-Time Linear Time-Invariant Systems."

The files require the installation of MOSEK and YALMIP:

MOSEK:  https://docs.mosek.com/10.2/toolbox/index.html

YALMIP: https://yalmip.github.io
